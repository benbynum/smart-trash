Smart Trash

To allow public deletion of posts for a project I'm working on (whether offensive, off-topic, or other) I created this Smart Trash directive. The idea is that three different users would have to click delete before the content gets deleted. This is to prevent a single user from having the power to wipe out all posts, while still allow for crowd sourced moderation. Depending on total users, the threshold may need to be higher than 3.
